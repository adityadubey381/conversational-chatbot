# Conversational Chatbot

A simple conversational AI chatbot built using transformer models, designed to generate human-like and emotionally aware responses.

---

## Features

* Context-aware responses
* Emotion-based conversation handling
* Easy to extend and modify

---

## Model Not Included

The trained model (`.safetensors`) is not included due to GitHub's 100MB file limit.

---

## Model Training (Colab)

The model was trained using Google Colab.
You can access the full training pipeline here:

**Link:-** https://colab.research.google.com/drive/1DlksLWJ3FESkLPJ6SMPQ3O8Xd65hSKpy?usp=sharing

## Setup

```bash
git clone https://github.com/adityadubey381/conversational-chatbot.git
cd conversational-chatbot
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

---

## Run Chatbot

```bash
python app.py
```

---

## Using a Model

To run the chatbot, you need a model:

* Train using the Colab notebook
  **or**
* Load a pretrained model from Hugging Face

* Python
